# Handoff

The handoff package should contain only the clean UI output.

## Include

- final UI pages
- shared assets needed by those pages
- documentation that helps a developer understand the structure

## Exclude

- wireframe version history
- encrypted preview payloads
- demo-only logic
- experimental branches or throwaway builds

## Handoff shape

The clean deliverable should be easy to package as:

- a folder
- a zip archive
- or a dedicated UI branch/output

The output should read like a finished implementation, not a work-in-progress playground.
